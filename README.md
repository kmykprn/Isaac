# Isaac
Isaac SimとIsaacLabのpoetry環境構築用。

## 必要なもの
Poetry (version 2.1.1)

## インストール
```
git clone https://github.com/kmykprn/Isaac.git
cd Isaac
poetry install
```

## 実行
```
poetry run isaacsim
```
